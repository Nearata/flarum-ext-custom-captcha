> Custom Captcha for Flarum

## Install

```sh
composer require nearata/flarum-ext-custom-captcha:"*"
```

## Update

```sh
composer update nearata/flarum-ext-custom-captcha:"*"
php flarum cache:clear
```

## Remove

```sh
composer remove nearata/flarum-ext-custom-captcha
php flarum cache:clear
```
